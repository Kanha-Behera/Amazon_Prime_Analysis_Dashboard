# Amazon_Prime_Analysis_Dashboard
Amazon Prime Content Analysis Dashboard is an interactive Power BI project that provides insights into Amazon Prime's content library through data visualization and business intelligence techniques. The dashboard analyzes content trends, genre distribution, ratings, release patterns and geographical availability to help users better understand.

**Overview**

This project is an interactive Power BI dashboard designed to analyze Amazon Prime's content library and uncover valuable insights related to content distribution, ratings, release trends, genres, and geographical availability.

The dashboard enables users to explore Amazon Prime data through dynamic visualizations and data-driven storytelling techniques.

🎯 Project Objectives
Analyze Amazon Prime's content catalog.
Identify trends in content releases over time.
Compare Movies and TV Shows distribution.
Explore content ratings and genre popularity.
Visualize country-wise content availability.
Create an interactive and user-friendly dashboard for business insights.
